## Software

Created using [Reveal.JS](https://github.com/hakimel/reveal.js) and [Jekyll-Reveal.JS](https://github.com/dploeger/jekyll-revealjs).

To run just clone the repo and do ```jekyll serve -H 127.0.0.1```
